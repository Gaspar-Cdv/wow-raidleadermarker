# RaidLeaderMarker

Currently, the crown icon for the raid or party leader is not displayed on raid frames. This lightweight addon fixes that to make it easier to identify the leader at a glance. Additionally, it also allows you to display the raid target marker icon on raid frames.

## Features

- Displays a crown icon on the raid frame of the current group leader.
- Displays the raid target marker icon on the raid frame.
- Works for any party (dungeon, raid, delve, etc.).
- Updates in real-time when the leader changes or when the raid target marker changes.
- Customizable icon size and position through the settings menu.
