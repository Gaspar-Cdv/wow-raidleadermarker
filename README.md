# RaidLeaderMarker

Currently, the crown icon for the raid or party leader is not displayed on raid frames. This lightweight addon fixes that to make it easier to identify the leader at a glance.

## Features

- Automatically displays a crown icon on the raid frame of the current group leader.
- Works for any party (dungeon, raid, delve, etc.).
- Updates in real-time when the leader changes.
